# Oogiri Evaluation Sheet

## 使い方
1. 1回答につき1行で記録する。
2. 回答文の良し悪しではなく、どの軸で当たった/外れたかを記録する。
3. 30件以上で集計し、キャラ別に弱い軸を補正する。

## 記録テンプレート
| date | topic | topic_type | speaker | answer | result | clarity(1-5) | surprise(1-5) | imagery(1-5) | brevity(1-5) | character_fit(1-5) | gaya_seed(1-5) | fail_reason | note |
|---|---|---|---|---|---|---:|---:|---:|---:|---:|---:|---|---|
| 2026-03-10 | 100年後の回転寿司 | あるある | 粗野 | ... | hit/close/weak/crash |  |  |  |  |  |  |  |  |

## fail_reason の候補
- `オチ弱い`
- `説明過多`
- `画が浮かばない`
- `既出と被り`
- `キャラ不一致`
- `文が長い`

## topic_type の候補
- `あるある`
- `写真で一言`
- `空欄穴埋め`
- `会話の続き`
- `商品コピー`
- `不条理一言`
